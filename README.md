# Pomodoro Kids

نسخه اولیه نرم‌افزار مدیریت زمان مطالعه، بازی و اینترنت برای نوجوانان با داشبورد ماینکرفتی.

## امکانات فعلی
- داشبورد الهام‌گرفته از Minecraft با پنل سه‌بخشی (پروفایل، تایمر حلقه‌ای، امتیاز)
- پروفایل‌های آماده برای مطالعه، بازی و اینترنت
- ویرایش تنظیمات هر پروفایل (زمان کل، تمرکز، استراحت، یادآور)
- اجرای جلسه و ثبت امتیاز هفتگی/ماهانه/سالانه
- نمایش جایزه بعدی و امتیاز باقی‌مانده تا باز شدن جایزه
- بررسی خودکار جوایز تعریف‌شده توسط والدین
- ذخیره‌سازی محلی در `data/app_state.json`

## اجرای برنامه (Windows)

```bash
pip install -r requirements.txt
python main.py
```

یا در ویندوز:

```bat
run_windows.bat
```

## اجرای تست‌ها

```bash
pytest
```

## رفع مشکل نمایش پنجره قدیمی
اگر هنوز پنجره ساده/قدیمی می‌بینید (مثل عنوان `Pomodoro Kids` بدون داشبورد)، معمولاً برنامه از build قدیمی (`dist/`) اجرا می‌شود.

1. پوشه‌های قدیمی را پاک کنید:
   - `dist/`
   - `build/`
2. برنامه را از سورس اجرا کنید:
   - `python main.py`
3. عنوان پنجره باید شامل نسخه UI باشد:
   - `Pomodoro Kids | UI-MC-v2`
